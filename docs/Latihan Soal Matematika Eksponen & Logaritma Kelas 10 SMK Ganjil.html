A. 16
B. 24
C. 32
D. 48
Lihat Pembahasan
Jawaban: A
a^(-1/3) = 27^(-1/3) = (3^3)^(-1/3) = 3^-1 = 1/3. b^(2/5) = 32^(2/5) = (2^5)^(2/5) = 2^2 = 4. 3(1/3) x 4(4) = 1 x 16 = 16.
